# 0.2 Beta Testing

Test only game and homebrew images you own or are authorized to use. Never
attach ROMs, saves, original firmware, private screenshots, or Translation Lab
evidence to a public report.

## Primary test areas

- BIOS-free startup on WonderSwan, WonderSwan Color, SwanCrystal, and Pocket
  Challenge V2 through SwanSong Open IPL;
- local direct game/homebrew import from supported files and one-game ZIPs;
- pause, reset, fast-forward, rotation, screenshots, saves, states, Time Ribbon,
  display profiles, and clean relaunch;
- standard GameController mapping, reconnection, multiple-device merge, and
  Debug Tools focus/input evidence; and
- Apple silicon and Intel behavior on macOS 14 or later.
- manual app-update checks, opt-in automatic checks and download/install,
  stable-only behavior, beta opt-in, cancellation, relaunch, and
  current-version/no-update behavior.

## Expected unavailable features

- The Homebrew page says **Coming Soon** and performs no catalog or game
  download. Use **Add From Mac**; direct GitHub installation is not enabled.
- The app contains no original-firmware import or override.
- The app does not install or update the separate Analogue Pocket core.
- A USB HID device that macOS does not expose through GameController may not
  appear and is not assigned a guessed raw mapping.

## Reporting checklist

1. Confirm version 0.2.0 and the intended build in About SwanSong.
2. Record Mac model, macOS version, architecture, and controller name when
   relevant.
3. State whether the issue reproduces with an open-source fixture.
4. Distinguish **Reached Video** from an end-to-end **Works** verdict.
5. For input/focus bugs, enable Debug Tools and review the source-free JSON log
   before exporting it.
6. Report expected and observed behavior without attaching private game data.
7. For an updater issue, include the installed and offered version/build,
   stable/beta selection, automatic-check/download choices, and exact visible
   error. Do not attach an archive that failed verification.

See [[Gamepads]], [[Open IPL]], [[Homebrew Catalog]], and [[App Updates]] for
exact boundaries.
