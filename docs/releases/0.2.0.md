## SwanSong 0.2.0 beta — Open by Default

This beta removes the last original-firmware workflow, strengthens controller
handling, and prepares a signed first-party homebrew installer without enabling
its network catalog prematurely.

### What changed

- SwanSong Open IPL is now the only startup path for WonderSwan, WonderSwan
  Color, SwanCrystal, and Pocket Challenge V2. The app has no BIOS picker,
  original-firmware import, storage, or override path.
- Controller discovery and remapping use the standardized controls that macOS
  exposes through GameController, including Extended, Micro, Directional, and
  bounded arcade-button aliases. Multiple connected controllers merge safely,
  and disconnecting one does not clear another controller's held input.
- Optional Debug Tools add a live focus/input overlay and a bounded,
  user-exported source-free input/frame log for controller troubleshooting.
- A signed first-party Homebrew Catalog installer, verified cache, anti-rollback
  state, bounded download path, and update-preserving library transaction are
  implemented behind a fail-closed production gate.
- A native Sparkle 2 app updater reads a signed feed from this repository and
  installs verified, immutable SwanSong Desktop GitHub Release assets. Manual
  checks remain available; automatic checks and automatic download/install are
  separate opt-ins, and beta updates require an independent channel choice.
- Release archives now have stricter source-payload, architecture, signature,
  notarization, checksum, manifest, installation, and rollback verification.
  Signed app metadata, the manifest, source-archive provenance marker, and
  both ares lock copies must agree on clean source and engine commits. The
  schema-v2 manifest and provenance marker also bind the archived Sparkle lock
  to its exact materialized Sparkle commit and corresponding source.

### Homebrew status

The Homebrew Catalog is **Coming Soon** in this beta. No production catalog
public key is embedded, so opening Homebrew makes no catalog or game-download
request. **Add From Mac** remains available for authorized local homebrew.
Direct installation from GitHub will be enabled only after a non-empty signed
catalog and immutable first-party release assets pass the release gate.

Sparkle does not power or bypass this catalog gate. Sparkle updates the macOS
application only; homebrew distribution retains its separate repository,
signature key, schema, consent, ROM validation, and publication state.

### Controller scope

USB and Bluetooth controllers are supported when macOS presents them through
GameController with standard direction and action inputs. This is not a claim
that every generic USB HID device works. SwanSong deliberately does not guess
raw vendor-specific button numbers or reports.

### Separate Analogue Pocket core

The Pocket build and its release authorization remain separately owned by
[`RegionallyFamous/swansong-core`](https://github.com/RegionallyFamous/swansong-core).
Desktop now includes an explicit **Analogue Pocket** card-preparation tool that
accepts only an immutable authorized stable Core release after its manifest,
asset size, and SHA-256 agree. It merges only the Core's managed files and does
not format the card or change games, saves, Memories, Settings, Presets, or
other cores. It checks free space, retains a recovery copy during the
transaction, and reads every managed file back before reporting success. No
verified Core release is currently published, so the tool reports that status
and performs no write.

### Before you install

SwanSong requires macOS 14 or later and supports Apple silicon and Intel Macs.
No games or original system firmware are included. Add only game and homebrew
images you own or are authorized to use.

This beta must be downloaded explicitly from the repository's Releases page;
the stable `/releases/latest` link and stable Sparkle channel remain on the
latest non-prerelease. Testers who already have an updater-capable build may
enable **Include beta versions** instead. Verify the universal app ZIP against
`SHA256SUMS.txt` and the accompanying release manifest before a manual install.

### Known limits

- Commercial-title compatibility is still being validated. **Reached Video**
  is not a full-game verdict or original-hardware accuracy claim.
- The Homebrew Catalog network and direct-install UI remain unavailable until
  production publication and trust material exist.
- Controllers that macOS does not expose through GameController require a
  compatible driver or mapping layer, or keyboard input.
- Automatic update checks and automatic download/install are off until the
  tester opts in. Sparkle system profiling is disabled.
