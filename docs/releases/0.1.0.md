## SwanSong 0.1.0 — First Flight

SwanSong is a private, native WonderSwan player and translation workbench for
macOS.

### Highlights

- Native WonderSwan, WonderSwan Color, SwanCrystal, and Pocket Challenge V2
  play through a pinned ares engine.
- Landscape and portrait presentation with integer scaling and four live
  display profiles.
- Keyboard and controller mapping for both WonderSwan direction clusters.
- Memory-only 30-second Time Ribbon and screenshot-backed save-state timeline.
- Private local game library and user-supplied startup-file validation.
- Conservative local Game Confidence evidence that never turns a rendered
  frame into a full compatibility claim.
- Optional local Translation Lab with deterministic route verification,
  evidence review, and manual capture-and-draft translation workspaces.

### Before you install

SwanSong requires macOS 14 or later and supports Apple silicon and Intel Macs.
No games or system startup files are included. Add only local copies you own or
are authorized to use.

Download `SwanSong-0.1.0-macOS-universal.zip`, expand it, and move SwanSong to
Applications. The release is Developer ID signed, notarized by Apple, and
distributed with SHA-256 checksums and exact corresponding source.

### Known limits

Commercial-title compatibility is still being validated. **Reached Video**
means SwanSong produced a meaningful game raster; it is not a full-game verdict
or an original-hardware accuracy claim. Updates are currently installed
manually from this Releases page.

### Privacy

SwanSong itself initiates no network requests. It has no accounts, ads,
analytics, telemetry, crash-reporting service, or automatic update checks.
Games, startup files, saves, states, screenshots, compatibility notes, and
translation evidence stay on your Mac unless you explicitly export them.
