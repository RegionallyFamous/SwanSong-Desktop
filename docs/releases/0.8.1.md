## SwanSong 0.8.1 — Keep the Shelf Clear

SwanSong 0.8.1 is a focused interface patch. Internal compatibility confidence
stays available to translators and testers through Developer Tools, while the
everyday library returns to the simple experience of choosing a game and
playing it.

Expected app identity: **SwanSong 0.8.1 (17)**.

### The short version

- Library cards no longer show **Picture Appeared**, **Not Tested Yet**, or
  **Needs Attention** unless Developer Tools is enabled.
- The complete Game Confidence panel, verdict controls, notes, and related
  accessibility descriptions follow the same setting.
- Play, repair, game details, and save controls remain available to everyone.
- Existing compatibility evidence is preserved when the developer surfaces are
  hidden.

### A quieter everyday library

Reaching a frame is useful diagnostic evidence, but it is not a promise that a
whole game works. Those labels belong with SwanSong's testing tools, not in the
normal player experience.

With Developer Tools off, game cards and the inspector now focus on the actions
needed to play and manage a library. Turning Developer Tools on restores the
compatibility badges, Game Confidence panel, verdict controls, and notes for
people doing deeper validation.

This is a presentation change only. It does not erase compatibility evidence,
change managed-game health checks, or hide a real repair problem.

### The complete 0.8 workshop remains

SwanSong 0.8.1 retains everything introduced in 0.8.0: Translation Surface
Suites, multi-checkpoint clean-boot routes, native-1× review, immutable
certification, faster observed-play authoring, Story Forge quality evidence,
and the visual signed Homebrew Catalog.

### Install or update

Existing users can choose **SwanSong > Check for Updates…**. A manual install
can download `SwanSong-0.8.1-macOS-universal.zip` from the GitHub Release and
verify it with `SHA256SUMS.txt` and `SwanSong-0.8.1-release.json`.

SwanSong requires macOS 14 or newer. It includes no games or original system
firmware. Use only software and hardware you own or are authorized to use.

If anything feels wrong, open **Help > SwanSong Support** and create a
privacy-safe Support Bundle. The [support guide](../../SUPPORT.md),
[privacy policy](../../PRIVACY.md), and
[complete changelog](../../CHANGELOG.md) have the deeper details.
