## SwanSong 0.9.0 — Bring the Translation Home

SwanSong 0.9 adds Translation Shelf: a local path from a trusted, source-free
fan-translation release to a verified, separately managed game in your
library. It also keeps agent playtesting prompts with the developer tools they
belong beside and refreshes the approved first-party Homebrew artwork.

Expected app identity: **SwanSong 0.9.0 (18)**.

This is the current stable release for supported Apple silicon and Intel Macs.

### The short version

- Choose a release-certified `release.json` package and the exact original
  game it requires.
- SwanSong verifies the manifest, IPS, input revision, finished-game digest,
  WonderSwan structure, cartridge checksum, and save/RTC contract.
- Patching happens in memory. The selected original is not changed or copied
  into the release package.
- The translated build receives its own library identity, badge, saves, states,
  artwork, favorite, and play history.
- An existing exact finished build can be adopted instead of duplicated.
- A missing or changed managed translated copy can be rebuilt from the same
  verified package and original without losing its library history.

### Trust boundary

Translation Shelf is source-free and network-silent. It does not search for or
download translations, original games, or patches. A local manifest is not a
publisher signature: get the package from a person or site you trust, then
SwanSong protects the byte-for-byte path from that package and exact original
to the managed translated game.

Unsafe paths, links, special files, oversized inputs, unsupported patch
formats, the wrong original revision, a changed IPS, an unexpected output, or
an invalid cartridge checksum all stop installation.

### Developer-only playtest prompts

With **Show developer tools** enabled, Homebrew includes a SwanSong Fun Tester
panel that can copy a structured Codex request for all ten SwanSong Originals
or the selected game. The banner, per-game card, and diagnostic wording remain
hidden in the normal player experience.

Seven approved Homebrew title screens have refreshed native captures, and the
production appcast publisher now requires a maintainer-approved protected
environment before it can use the signing secret.

### Install the release

Existing users can choose **SwanSong > Check for Updates…** on the default
stable channel. A manual install can download
`SwanSong-0.9.0-macOS-universal.zip` from the GitHub release and verify it
with `SHA256SUMS.txt` and `SwanSong-0.9.0-release.json`.

SwanSong requires macOS 14 or newer. It includes no games, translations, or
original system firmware. Use only software and hardware you own or are
authorized to use.
