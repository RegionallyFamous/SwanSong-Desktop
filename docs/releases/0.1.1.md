## SwanSong 0.1.1 — Clear Signals

This patch release makes SwanSong's identity and support surfaces immediately
understandable on macOS.

### What changed

- A new swan app icon replaces the previous mark that could resemble a rocket
  at small menu and Dock sizes.
- SwanSong Open IPL now starts every supported WonderSwan model without asking
  for an original BIOS; authorized original firmware is an optional private
  compatibility override.
- Privacy, Support, License, and Acknowledgements now open together in a native
  Legal & Support window instead of handing Markdown and text files to another
  application.
- The Help menu now includes native Help, Check for Updates, and Report a
  Problem entries.
- Support information can be copied without including game names, private
  paths, ROM data, saves, or translation content.
- Startup-file storage is explicitly revealed in Finder.

### Before you install

SwanSong requires macOS 14 or later and supports Apple silicon and Intel Macs.
No games or original system firmware are included. Open IPL is independently
written project code. Add only game and optional firmware copies you own or are
authorized to use.

Download `SwanSong-0.1.1-macOS-universal.zip`, expand it, and move SwanSong to
Applications. The release is Developer ID signed, notarized by Apple, and
distributed with SHA-256 checksums and exact corresponding source.

### Privacy and updates

SwanSong still initiates no network requests. It has no accounts, ads,
analytics, telemetry, crash-reporting service, or background update checks.
Choosing **Open SwanSong Releases** or **Report a Problem** is an explicit
action that opens the Regionally Famous GitHub project in your default browser.

### Trust and verification

- Bundle identifier: `com.regionallyfamous.swansong`
- Developer ID Team: `3J8H48TP7P`
- Universal architectures: `arm64`, `x86_64`
- Minimum system: macOS 14.0

Use `SHA256SUMS.txt` to verify both the app and source archives. The accompanying
