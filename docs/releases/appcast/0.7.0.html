A safer SwanSong—without more fuss.

SwanSong 0.7 puts the game engine in its own locked-down service, hardens local automation, adds a launch-rescuing Safe Mode, and creates privacy-safe Support Bundles.

  • Play behind a stronger boundary. The isolated engine accepts only typed, size-limited data and has no network access.
  • Trust local automation. Fresh, one-use requests must come from SwanSong's signed helper and the current Mac user.
  • Get unstuck. Safe Mode skips restored work and background helpers without touching your library or saves.
  • Ask for help safely. Support Bundles leave out games, saves, screenshots, projects, private paths, and accounts.
  • Trace translations farther. Deeper source provenance stays private while MCP receives only counts, hashes, geometry, and completeness.

Every release now includes exact source, checksums, an SPDX software bill of materials, and verifiable artifact attestations.

SwanSong includes no games or original system firmware. Read the full release notes.
