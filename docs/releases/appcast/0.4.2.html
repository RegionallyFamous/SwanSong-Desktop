Make the next game smaller, clearer, better.

SwanSong 0.4.2 brings the released SwanSong SDK 0.3.1 toolset into Studio, with native controls for visual authoring, replay inspection, and deterministic failure reduction.

  • Shape the game. Create, validate, report, and export tilemap, sprite, palette, collision, scene-flow, and audio documents.
  • Read the replay. Turn exact-frame plans, checkpoints, evidence, and traces into one read-only timeline.
  • Shrink the failure. Delta-reduce a failing plan through bounded fresh SwanSong runs and verify the failure still holds.
  • Trust the bundle. Studio content-verifies the exact tagged SDK runtime, schema, recipes, docs, and Python package.
  • Finish Doctor cleanly. Its bounded local SwanSong probe no longer waits for a valid persistent server to exit.

Python and Wonderful remain local prerequisites, and authoring or replay reports do not replace inspected gameplay evidence. Read the full release notes.
