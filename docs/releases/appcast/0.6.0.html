The whole SwanSong workshop can move with you.

SwanSong 0.6.0 brings the complete SDK 0.5 Studio, a much richer Story Forge, and frame-authenticated Translation Lab automation to the public Mac app.

  • Build more in Studio. Create Utility Apps, run traced builds and scenarios, inspect outcomes and budgets, migrate projects, and finish an evidence-backed release.
  • Write in a real Story Room. Map the story, edit chapters, preserve revisions, work with unprimed readers and research, audition art and music, and prepare a source-mapped WonderSwan adaptation.
  • Trust the exact captured frame. Capture-bound source probes validate the private native frame before provenance begins and leave nothing partial behind on a mismatch.
  • Enjoy calmer local tools. Trusted automation no longer needs to touch the login keychain, and release builds gain a sealed, reproducible engine gate.

SwanSong includes no games or original system firmware. Studio and Story Forge use separately installed development tools or selected local projects. Read the full release notes.
