Translation Lab can follow the pixels.

SwanSong 0.7.1 gives source probing a signed, capture-authorized path from a visible rectangle back to its real cartridge lineage—while private game details stay inside your project.

  • Probe the frame you captured. The project, ROM, plan, frame, rectangle, helper, runner, and engine must all agree.
  • Change with confidence. Select map, raster, palette, or sprite ownership and still discover other display elements sharing the same source.
  • Keep private evidence private. Automation receives safe counts, hashes, geometry, and completeness—not addresses, cartridge ranges, mapper state, or source bytes.
  • Trust the finished proof. SwanSong writes the closure last, reads it back, and validates the complete evidence tree.
  • Skip the password box. Runtime and local-tool release gates keep ordinary SwanSong use away from the login Keychain.

SwanSong includes no games or original system firmware. Read the full release notes.
