Bring a trusted translation home to your library.

SwanSong 0.9 adds a local Translation Shelf for source-free, release-certified IPS packages.

  • Verify the exact path. SwanSong checks the package, IPS, required original revision, finished-game digest, cartridge checksum, and save contract.
  • Keep the original unchanged. Patching happens in memory, and only the verified translated result enters the private managed library.
  • Keep histories separate. The translated game receives its own identity, saves, states, artwork, favorite, and play history.
  • Repair without starting over. Rebuild a missing or changed managed copy while preserving its library history.
  • Developer tools stay developer-only. Agent playtest prompts and diagnostic confidence language remain hidden unless Developer Tools is enabled.

SwanSong 0.9.0 is the current stable release. It is offered on the default update channel.

SwanSong includes no games, translations, or original system firmware. Read the full release notes.
