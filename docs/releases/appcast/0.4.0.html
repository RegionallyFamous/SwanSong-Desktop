Build where you play.

SwanSong 0.4 brings the SDK workflow into a native Game Studio and gives Translation Lab a deeper, private path from visible pixels to cartridge sources.

  • Make a game in SwanSong. Create, edit assets, build, test, play deterministic contracts, and review resource reports in one workspace.
  • Trace upstream. Follow a selected rectangle through copies and transforms while exact cartridge details stay private.
  • Use the whole toolset. The installed plugin exposes persisted capture, both probes, record/verify, and the complete observed-play lifecycle.
  • Find the swan. SwanSong now has its own compact menu-bar status item.

Game Studio is a developer preview that currently uses a separately installed SDK checkout and toolchain. Read the full release notes.
