Build a WonderSwan game without leaving SwanSong.

SwanSong 0.4 adds Studio, a focused home for the complete SwanSong SDK workflow.

  • Make and iterate. Create a recipe, build assets, test, play, profile, and package from one native workspace.
  • See problems sooner. Doctor, the input fuzzer, Sprite/VRAM profiler, Asset Optimizer, and Save/RTC Laboratory expose setup, gameplay, and hardware-budget failures.
  • Turn play into proof. Record input scenarios, compare visual and audio evidence, and require inspected results before release.
  • Keep one source of truth. Studio delegates to SwanSong SDK 0.2 or newer and uses SwanSong as its only gameplay backend.

Your projects, ROMs, diagnostics, and evidence stay on your Mac. Read the full release notes.
