Build where you play.

SwanSong 0.4 adds Studio for the complete SwanSong SDK workflow and gives Translation Lab a deeper, private path from visible pixels to cartridge sources.

  • Make and iterate. Create a recipe, build assets, test, play, profile, inspect evidence, and package from one native workspace.
  • Turn play into proof. Record deterministic scenarios, compare visual and audio evidence, and require inspected results before release.
  • Trace upstream privately. Select map, raster, or palette provenance while exact cartridge and executed-read details stay inside the project.
  • Use the whole toolset. The installed plugin exposes persisted capture, both probes, record/verify, and the complete observed-play lifecycle.
  • Find the swan. SwanSong now has its own compact menu-bar status item.

Studio is a developer preview that uses SwanSong SDK 0.2 or newer and a separately installed toolchain. Your projects, ROMs, diagnostics, and evidence stay on your Mac. Read the full release notes.
