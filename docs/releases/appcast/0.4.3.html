Every press reaches the game.

SwanSong 0.4.3 makes exact-frame controls reliable, brings SwanSong SDK 0.4.0 into Studio, and makes the app's main screens easier to understand.

  • Trust short inputs. Presses and releases reach the live WonderSwan keypad on their scheduled frame.
  • Build on SDK 0.4.0. Studio content-verifies the released runtime, schema, recipes, docs, and Python package.
  • Keep games responsive. Normal frame presentation no longer performs expensive whole-map and whole-sprite profiler scans.
  • Find the next action. Library, Homebrew, Translation Lab, Studio, Settings, Updates, and Support lead with clearer language.

Python and Wonderful remain local prerequisites. SwanSong includes no games or original system firmware. Read the full release notes.
