SwanSong is ready before you are.

SwanSong 0.6.1 opens faster, keeps an old catalog trust record from interrupting launch, and helps Studio find the modern Python already installed on your Mac.

  • See your library sooner. The main window appears first while controllers, local tools, and the workshop finish waking up behind it.
  • Skip the surprise password box. Homebrew Catalog trust checks now fail quietly instead of asking for the Mac login password.
  • Open Studio from Finder. SwanSong finds Python 3.11+ in standard Homebrew, python.org, MacPorts, and system locations.
  • Keep everything from 0.6. Studio, Story Forge, Translation Lab, Time Ribbon, save states, and the native player are all here.

SwanSong includes no games or original system firmware. Development tools remain separate prerequisites. Read the full release notes.
